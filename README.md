# WPSM applicant test

We’ve added some inconsistencies to the projects. The errors can be of (programming-) technical, functional or security related nature. 

## Getting Started

* You should open the solution with Visual Studio 2017
* The solution contains 5 projects. The projects `DevTestCon` and `DevTestWeb` need to be setup as start projects.
* `DevTestCon` will open the port 27771 as a RemotingPort.
* Afterwards the `DevTestWeb` can be opened via browser.

### Prerequisites

* .NET Framework 4.6.2
